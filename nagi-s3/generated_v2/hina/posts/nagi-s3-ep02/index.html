午後の会議室。真壁と古参が並び、腕を組んでいる。
「E2E を回してきたから、ここまで来れたんです。ユニット？ API？理屈は分かるけど、現場は時間ないんで」
誰かが言うと、頷きが連鎖した。旧来 GUI 自動化教団――そう揶揄される彼らの結束は固い。

ナギは一度も声を荒げない。ホワイトボードに三角形を描いた。
底辺に「Unit / API」、中段に「Service」、頂点に「GUI」。

「テストピラミッドです」
真壁が鼻で笑う。「教科書っすね」
「教科書は、死なないから便利なんですよ」
ナギはさらりと言い返すと、頂点を指した。

「全部 GUI でやるのは、山の頂上に全員を住まわせるようなものです。高いし、崩れやすいし、寒い」
古参が身じろぎする。自分たちの“成果”を寒いと言われた気がしたのだ。

ナギは三角形の頂点に小さく丸を付けた。「GUI は代表サンプルだけ。ここは“儀式”じゃなく“証拠”の層です」
続けて底辺を太く塗る。「安い検証を増やす。速い検証を増やす。壊れにくい検証を増やす」

「でも、ユーザーの操作は GUI でしか……」
真壁が口を開くと、ナギは頷いた。
「だから代表を残す。だけど、代表以外まで GUI で抱える理由は、怖さ以外にありますか？」

沈黙。ミオは喉の奥が熱くなるのを感じた。怖さ以外――それが言語化できない。
ナギは会議室の空気を急かさない。代わりに宿題を出す。

「次回までに、E2E 300 本を三つに分類してください」
①“ユーザーの行動証拠として必要” ②“仕様の制約確認として必要” ③“惰性で残ってる”
「分類できないなら、なぜ分類できないかを書いてください。それが、世界 W の霧です」

会議が終わり、皆が出ていく。最後にミオが立ち上がりかけたとき、ナギが静かに声をかけた。
「早瀬さん、ノート、真っ黒ですね」
「……すみません、勝手に」
「いい。図は写すと、体に入る。次は自分で描けるようにしよう」

ミオは頷いた。真壁は背中越しに小さく舌打ちしたが、その音は以前ほどの確信を持っていなかった。
