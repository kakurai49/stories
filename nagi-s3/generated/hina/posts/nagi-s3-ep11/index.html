別チームが合流してきた。新規案件を抱えた彼らは、開口一番こう言った。
「E2E、増やしましょう。安心ですし」
ミオは反射的に身構える。昔の自分なら、同じことを言っていた。

しかし返答したのは、真壁だった。
静かに、ホワイトボードに三角形を描く。テストピラミッド。
「うちはもう、こういう方針なんで。GUI は代表だけ。あとはユニットと API でやります」
合流チームのリーダーが戸惑う。「でも、それでユーザー操作の保証は…」
真壁は続けて式を書く。
G : (W,E) → (S,A(S))
「保証したいのは操作じゃなくて、観点 S。世界 W と制約 E を見て、代表を選ぶ。テストを増やすのは最後の手段です」

ミオは息を止めて見ていた。
かつて「今まで通りでいい」と言っていた本人が、今は“新しい標準”を守っている。

会議が終わり、ナギは廊下でミオに言った。
「文化が変わったね」
ミオは頷く。「真壁さんが言うなんて…」
ナギは笑った。「真壁さんが変えたんだよ。私は言葉を渡しただけ」

その夜、真壁がミオに声をかけた。
「観点カード、レビュー頼む。俺、最近バイアス出る」
ミオは驚く。「私が？」
「おう。お前の方が、ユーザーの困り方を見てる」

それは権限の移譲ではない。視点の尊重だ。
“設計”がチームの言葉になり、“怖さ”がチームの共通課題になったとき、旧来の信仰は自然に溶けていく。
ナギは遠くからそれを見て、ひとつだけ確信する。
次の現場へ行ける、と。
