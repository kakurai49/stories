第10話「ミオの初陣 ― 私の G を描く」

新機能の開発が始まった。ナギは一歩引き、ミオに任せる。
「今回のテスト設計、あなたが主担当。私はレビューだけ」
ミオの心臓が跳ねる。嬉しさより、責任の重さが先に来た。

ホワイトボードの前。ミオは深呼吸し、書き始める。
W：ユーザー属性（新規／既存）、端末種別、ログイン状態、通信状況
E：法規制（年齢／地域）、ビジネス制約（無料枠、課金導線）、性能（タイムアウト）
S：観点（成功、拒否理由の違い、境界、UX、リカバリ）

そして、式を書く。
G : (W,E) → (S,A(S))

「観点を決めるとき、怖さで増やさない。困り方で分ける」
ミオは自分に言い聞かせるように呟く。真壁が横で黙って見ていた。以前なら口を出したはずだ。

ミオは MBT の図を描き、代表パスを選ぶ。GUI は 2 本。残りは API とユニットへ落とす。
レンズは、ログインと決済をまたぐため、WorldState の境界が難しい。ミオは悩み、ナギにだけ小声で聞く。
「レンズが大きくなりすぎそうで…」
ナギはすぐ答えない。代わりに聞き返す。
「そのレンズは、何を隠したい？ 何を見せたい？」
ミオは考え、言い直す。
「見せたいのは“ユーザーが何を選んだか”。隠したいのは“画面遷移の都合”」
「なら、選択を WorldState に記録して、遷移はレンズに閉じ込める」

設計が決まった瞬間、ミオの迷いが消える。
実装後、レビュー。ナギが頷いた。
「うん。“私の G”じゃなくて、“ミオの G”になってきたね」

数日後、ミオのテストが、リリース前に致命的な UX バグを拾った。キャンセル導線がユーザーを迷わせる。
真壁が驚いて言う。「そこ、誰も見てなかった」
ミオは小さく笑う。「S に入れました。困り方が変わるので」

怖さではなく、観点で守る。
その日、ミオは初めて“設計で殴る”感触を、自分の拳で確かめた。

「うちはもう、こういう方針なんで。GUI は代表だけ。あとはユニットと API でやります」
合流チームのリーダーが戸惑う。「でも、それでユーザー操作の保証は…」
真壁は続けて式を書く。
G : (W,E) → (S,A(S))
「保証したいのは操作じゃなくて、観点 S。世界 W と制約 E を見て、代表を選ぶ。テストを増やすのは最後の手段です」

ミオは息を止めて見ていた。
かつて「今まで通りでいい」と言っていた本人が、今は“新しい標準”を守っている。

会議が終わり、ナギは廊下でミオに言った。
「文化が変わったね」

第12話（最終話）「次の現場へ ― それぞれの G」

大きなリリース当日。深夜のオフィスに、以前のような悲鳴はなかった。
Jenkins は緑が多い。落ちても原因がすぐ分かる。WorldState のログが、迷子を作らない。
GUI テストのメンテは 5 時間で終わった。かつての 30 時間が、遠い悪夢みたいだ。

CPO がチャットに書き込む。「リリース通過。お疲れ」
CFO は短く返す。「数字、出てます。良い」
ミオは椅子にもたれ、肩から力が抜けるのを感じた。安心が、“回した本数”ではなく、“設計の地図”から来ている。

送別会の夜。小さな居酒屋。真壁が珍しくグラスを上げる。
「……神崎さん、最初はムカついたっす。怖いのを、数字で殴ってくるから」
笑いが起きる。ナギはグラスを軽く合わせた。
「怖さを責めたことはないよ。ただ、怖いまま増やすのは、チームを壊すから」

ミオが真剣な顔で聞く。
「ナギさん、次の現場でも G とかレンズとかやるんですか？」
ナギは少し考えてから答える。
「もちろん。でも、形は毎回変わる。世界 W と制約 E が違えば、G も変わる。それを考えるのが、テストアーキテクトの仕事だからね」

帰り道、ミオは駅のホームでナギに小さな箱を渡した。新品のホワイトボードマーカー。
「これ、私の初陣のときに使ったやつと同じ型です」
ナギは受け取り、目を細める。「次は、あなたが誰かに渡す番だ」

翌朝。ナギはオフィスのデスクを空にし、ノート PC を閉じた。
画面には、最後の資料が開いたまま。
G : (W,E) → (S,A(S))
その隣に、レンズの図。

エレベータの扉が閉まる直前、真壁が頭を下げた。
「……ありがとうございました。今まで通り、じゃダメなんですね」
ナギは微笑む。「“今まで通り”は便利だけど、世界は変わるから」

扉が閉まり、静かに下降する。
炎上の現場は、もう背中の向こうだった。

エピローグ「白いボードは、どこへでも」

半年後。ミオは、会議室のホワイトボードに三角形を描いていた。
新卒が不安げに聞く。「GUI テスト、もっと増やした方が安心じゃ…」

「安心は“増やす”じゃなく“見える”から来るよ。世界 W と制約 E を見て、観点 S を決める。代表を選ぶ。迷ったら地図を描く」

真壁が横で頷く。彼は今、観点カードのテンプレを整備し、レビュー会を回している。
CFO は月次でメンテ工数の推移を見て、静かに満足していた。数字は嘘をつかない。文化もまた、数字に支えられる。

その頃、別の街の別のビル。
神崎ナギは新しい会議室の扉を開け、真っ赤なダッシュボードを一瞥した。落ちまくる E2E、怯える目、そして聞き慣れた台詞。
「今まで通りで動いてるから、いいじゃん」

ナギはマーカーを受け取り、白いボードに最初の線を引く。
三角形。テストピラミッド。

G : (W,E) → (S,A(S))

「世界 W と制約 E をちゃんと見よう」
彼女の声は柔らかい。だが、その柔らかさは逃げ道ではなく、覚悟の形だった。

“今まで通り”に負けないために。
